# AjoTopup

AjoTopup adalah aplikasi web portal berbasis ASP.NET MVC untuk mengelola transaksi top-up pulsa telekomunikasi. Aplikasi ini menghubungkan operator/admin dengan gateway API yang terintegrasi dengan provider telekomunikasi (simulator provider dalam pengembangan).

## 🚀 Fitur Utama
- **Autentikasi Multi-Role**: Admin, Operator, dan Manager.
- **Manajemen Produk Pulsa**: CRUD data produk pulsa (Admin).
- **Pemrosesan Transaksi**: Pengisian pulsa real-time dengan tracking status.
- **Dashboard & Analytics**: Grafik transaksi per jam dan rasio sukses/gagal.
- **Log Request & Response**: Pemantauan histori API terperinci (Admin).

## 📂 Struktur Proyek
Proyek ini berisi modul-modul berikut:
- **AjoTopup.csproj**: Aplikasi Web Portal utama menggunakan ASP.NET MVC 5.
- **Database_Setup.sql**: Skrip inisialisasi basis data SQL Server.
